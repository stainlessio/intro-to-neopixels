# Hooking up

1. Trinket VBAT+ pin to 5V
2. Trinket GND to GND
3. Trinket RST to #10
4. Trinket #0 to #11
5. Trinket #1 to #12
6. Trinket #2 to #13

# Using

* Serial Monitor to 9600
* USB to arduino (not trinket)
* press `G` when prompted.
* only takes a few seconds
